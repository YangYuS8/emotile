# 路线图

## v0.1（已完成）

- 表达式验证、归一化、修复
- 纯数据渲染器（PixelFrame）
- 确定性变异
- ASCII 调试预览
- CommonJS 包
- 固定 32×32 画布

## v0.2（发布候选准备）

- 主题和调色板支持（设计提案）
- 纯数据动画 tick API
- 面向 Agent 的表达式辅助文档

## 延期

- 可变画布尺寸（超出 32×32）
- 主题和调色板运行时实现
- ESM / 双构建
- 浏览器 / Canvas / GPU 集成
- 预设表情目录
- PNG / SVG / GIF 导出
- 终端渲染器（盲文/unicode）
- 游戏引擎集成
