# 路线图

## v0.1（已完成）

- 表达式验证、归一化、修复
- 纯数据渲染器（PixelFrame）
- 确定性变异
- ASCII 调试预览
- CommonJS 包
- 固定 32×32 画布

## v0.2（已发布）

- 主题和调色板支持（设计提案）
- 纯数据动画 tick API
- 面向 Agent 的表达式辅助文档
- Agent 生成指导和常见错误参考

## v0.3（规划中）

- 外部主题 / 调色板运行时
- 从 `PixelFrame` 到 SVG 字符串的 SVG renderer
- 用于验证、修复、预览和 SVG 渲染的轻量 CLI
- 用于 Agent 结构化输出的 JSON Schema 导出
- 展示可组合表达式的 gallery examples，但不做预设目录

## 延期

- 可变画布尺寸（超出 32×32）
- ESM / 双构建
- 桌面宠物运行时
- Canvas / 浏览器运行时
- 可视化编辑器
- 浏览器 / GPU 集成
- 预设表情目录
- PNG / GIF native 导出
- Live2D
- 大型依赖
- 终端渲染器（盲文/unicode）
- 游戏引擎集成
